# english-study

중간고사 영어 공부를 위한 정적 HTML 학습 페이지입니다.

이 저장소는 별도 빌드 도구 없이 브라우저에서 바로 열 수 있는 두 개의 화면으로 구성되어 있습니다.

## 무엇이 들어 있나

- `index.html`
  - 앱의 메인 진입 화면
  - 학습 시작 링크 제공
  - 전체 단어장 모드 제공
  - Day 선택 드롭다운, 단어장 보기, 퀴즈 모드 포함
  - 선택한 Day 기준 4지선다 뜻 맞히기 퀴즈와 오답 재도전 흐름 포함

- `midterm/data/school/interactive-study.html`
  - 실제 수업형 학습 화면
  - 회차별 단어, 예문, 학습 흐름, 퀴즈 UI 포함

## 현재 사용자 흐름

1. 메인 화면에서 학습 시작 또는 전체 단어장으로 이동
2. 전체 단어장에서 Day를 선택
3. 단어장 보기에서 단어를 눌러 뜻 확인
4. 퀴즈 모드에서 영단어를 보고 뜻을 고르는 객관식 풀이
5. 즉시 정오답 확인
6. 완료 후 정답률 확인, 틀린 단어만 다시 풀기 가능

## 실행 방법

별도 설치는 필요 없습니다.

다음 중 편한 방법으로 실행하면 됩니다.

1. `index.html`을 브라우저에서 직접 열기
2. 간단한 정적 서버로 현재 폴더 열기

예시:

```bash
cd /Users/kioku/Documents/Settings/english-study
python3 -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000` 접속

## 기술 스택

- HTML
- CSS
- Vanilla JavaScript

프레임워크나 번들러 없이 한 파일 안에 UI와 데이터를 함께 두는 방식으로 구성되어 있어 수정이 빠르고 배포가 단순합니다.

## 수정 포인트

- 단어 데이터 변경: 각 HTML 파일 내부의 데이터 배열 수정
- 단어장/퀴즈 동작 변경: `index.html`의 스크립트 영역 수정
- 수업 화면 구성 변경: `interactive-study.html` 수정

## 용도

이 프로젝트는 중간고사 영어 단어와 학습 흐름을 모바일 카드형 UI로 빠르게 복습할 수 있게 만드는 데 초점을 둡니다.
