# 📝 중간고사 동사 공부

> 고등학교 영어 중간고사 대비 — 49개 불규칙동사를 플래시카드와 퀴즈로 마스터하세요!

**👉 [지금 바로 시작하기](https://winterrainlee.github.io/verb-quiz/)**

---

## ✨ 주요 기능

### 📖 단어장 모드
- 49개 동사를 **플래시카드** 형태로 학습
- 카드를 터치하면 뒤집히며 **뜻, 3인칭 단수, 과거/과거분사, 힌트** 확인
- 이전/다음 버튼으로 자유롭게 탐색

### 🧠 퀴즈 모드
- 뜻과 힌트를 보고 **현재형, 과거형, 과거분사형**을 각각 입력
- 항목별로 ✅ 정답 / ❌ 오답 즉시 피드백
- Enter 키로 입력 칸 자동 이동
- 점수에 따른 **4단계 결과 메시지** (100% / 80%↑ / 50%↑ / 50%↓)
- **틀린 단어 확인** 및 **틀린 문제만 다시 풀기** 기능

### 🎨 테마
- ☀️ **라이트 모드** — 산뜻한 민트 그린
- 🌙 **다크 모드** — 세련된 코스믹 퍼플
- 기기 설정 자동 감지 & 선택 기억
