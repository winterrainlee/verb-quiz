## 핵심 전치사 12선

| **#**  | **전치사**       | **핵심 이미지 (방향성)**    | **볼링/축구 비유 ⚽️🎳**     |
| ------ | ------------- | ------------------- | --------------------- |
| **1**  | **TO**        | **화살표 (Target)**    | 핀을 향해 쭉! (도착지)        |
| **2**  | **FROM**      | **출발점 (Origin)**    | 파울 라인에서 출발! (시작점)     |
| **3**  | **AT**        | **점 (Point)**       | 에임 스팟 딱 찍기! (정확한 지점)  |
| **4**  | **IN**        | **박스 (Container)**  | 볼링장 건물 안! (공간 내부)     |
| **5**  | **ON**        | **찰떡 (Surface)**    | 레인 바닥에 딱 붙어서! (접촉)    |
| **6**  | **FOR**       | **목표/선물 (Goal)**    | 스트라이크를 위해서! (목적/대상)   |
| **7**  | **WITH**      | **동행 (Together)**   | 볼링공이랑 같이! (도구/동반)     |

| **11** | **UP / DOWN** | **위 / 아래**          | 점수판 위로! / 도랑 아래로!     |
| **12** | **OUT (OF)**  | **탈출 (Exit)**       | 레인 밖으로 나가기! (이탈/밖)    |
## 핵심 접속사 12선

| **#**  | **접속사**      | **한국어 뜻** | **철수 뇌 박제용 비유 (Vibe)**          |
| ------ | ------------ | --------- | ------------------------------- |
| **1**  | **AND**      | 그리고       | **1+1 행사:** 정보 하나 더 추가! (➕)     |
| **2**  | **BUT**      | 하지만       | **역전 홈런:** 분위기 확 바뀜! (🚫)       |
| **3**  | **OR**       | 또는        | **양자택일:** 짜장면 아님 짬뽕! (❓)        |
| **4**  | **SO**       | 그래서       | **인과응보:** 앞에 거 해서 뒤에 거 됨! (➡️)  |
| **5**  | **BECAUSE**  | ~때문에      | **이유 설명충:** 왜 그랬는지 말해줌! (📢)    |
| **6**  | **WHEN**     | ~할 때      | **타이밍:** 그때 딱 무슨 일이 생김! (⏰)     |
| **7**  | **IF**       | 만약 ~라면    | **가정 설정:** 혹시 모르니까 조건 걸기! (⚠️)  |
| **8**  | **ALTHOUGH** | ~이지만      | **반전 주의:** 그럼에도 불구하고! (끈기)      |
| **9**  | **WHILE**    | ~하는 동안    | **멀티태스킹:** 두 가지 일이 동시에! (🎥)    |
| **10** | **AS**       | ~해서, ~처럼  | **변신술사:** 뜻이 많지만 '때문에/할 때'가 기본! |
| **11** | **BEFORE**   | ~하기 전에    | **과거 회상:** 이 일 하기 전 상황! (⏪)     |
| **12** | **AFTER**    | ~한 후에     | **미래 전개:** 이 일 하고 난 다음 상황! (⏩)  |
