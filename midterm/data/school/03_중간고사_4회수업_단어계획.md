# 중간고사 대비 4회 수업 단어 계획

목표: [학교-단어장.md](/home/winterrain/projects/verb-quiz/midterm/data/school/학교-단어장.md)에 나오면서, [02_올림포스_지문_분류.md](/home/winterrain/projects/verb-quiz/midterm/data/school/02_올림포스_지문_분류.md)의 시험 범위와 밀접한 단어를 익혀서 학생이 지문의 대략적인 뜻을 이해하게 한다.

학생 수준: [철수의 단어장.md](/home/winterrain/projects/verb-quiz/midterm/data/철수의%20단어장.md) 기준의 기초 수준

운영 원칙

- 과학 지문은 과감하게 포기하고 `경제-심리-사회-인문` 중심으로 간다.
- 매 회차는 `쉬운 단어 감각 -> 학교 단어장 단어 -> 시험 지문 키워드` 순으로 연결한다.
- 목표 점수는 40~50점이므로, 정답률이 나오는 빈출 축을 우선 공략한다.
- 구성은 `Idea 1: 글 이해용 단어 2회`, `Idea 2: 특정 주제 단어 2회`로 잡는다.

## 전체 흐름

1. 1회차: 경제 지문 읽기 기본 어휘
2. 2회차: 심리·사회 지문 읽기 기본 어휘
3. 3회차: 심리 주제 단어 집중
4. 4회차: 사회·인문 주제 단어 집중

## 설계 메모

- 1회차는 가장 쉬운 경제 축으로 진입해서 학생에게 "지문도 읽을 수 있다"는 감각을 준다.
- 2회차와 3회차는 시험에서 비중이 큰 심리·사회 축으로 점수를 확보한다.
- 4회차는 문과 학생에게 유리한 사회·인문 지문 대응용 어휘로 마무리한다.
- 과학 지문 대비 어휘는 의도적으로 제외했다.
