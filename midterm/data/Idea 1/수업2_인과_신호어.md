## ⚡ 워밍업 — 이미 아는 것에서 시작 (5분)

| 기초 단어 (이미 앎) | 뜻 |
|---|---|
| **because** | ~때문에 |
| **so** | 그래서 |
| **cause** | 일으키다 |
| **make** | ~하게 만들다 |
| **reason** | 이유 |

👉 오늘 배울 단어들은 이것들보다 훨씬 **정교하게** 원인과 결과를 설명해.
영어 지문에서 인과 관계가 나오면, 반드시 이 단어들 중 하나가 등장해.

---

## 📚 오늘의 핵심 단어 10개

| # | 단어 | 품사 | 뜻 | 기초 단어 연결 | 뉘앙스 포인트 |
|---|---|---|---|---|---|
| 1 | **consequence** | 명사 | 결과, 영향 | `so` 뒤에 오는 것 | 주로 부정적·중요한 결과 |
| 2 | **contribute** | 동사 | 기여하다, ~의 원인이 되다 | `cause`의 긍정적 버전 | 직접 원인보다 한 요인으로서 |
| 3 | **trigger** | 동사/명사 | 촉발하다; 방아쇠 | `cause`의 갑작스러운 버전 | 총 방아쇠처럼 즉각적 |
| 4 | **induce** | 동사 | 유발하다, 유도하다 | `make` + 의도적 | 의도적으로 유도하는 느낌 |
| 5 | **derive** | 동사 | 끌어내다, ~에서 비롯되다 | `come from`의 격식체 | 기원·출처 강조 |
| 6 | **facilitate** | 동사 | 촉진하다, 용이하게 하다 | `help` + `cause` | 방해 제거해서 더 쉽게 |
| 7 | **compel** | 동사 | 강요하다, ~하지 않을 수 없게 하다 | `make` + 강제 | 선택의 여지 없이 |
| 8 | **impose** | 동사 | 부과하다, 강요하다 | `give` + 강제 | 외부에서 강제로 얹는 느낌 |
| 9 | **result in** | 구동사 | ~라는 결과를 낳다 | `so`의 동사 버전 | 결과가 뒤에 옴 |
| 10 | **stem from** | 구동사 | ~에서 비롯되다 | `because`의 동사 버전 | 원인이 뒤에 옴 |

---

### 💡 인과 방향 정리 — 헷갈리지 않게!

```
[원인] ──────────────────────► [결과]

trigger / induce / compel / impose / facilitate / contribute to
         (이 단어들 뒤에 결과가 온다)

consequence / result in
         (이 단어들 = 결과라고 직접 알려줌)

derive from / stem from
         (이 단어들 뒤에 원인이 온다 — 방향 반대 주의!)
```

## 🎯 독해 전략 정리 (5분)

### 수능 지문에서 인과 구조 찾는 법

**Step 1** — 인과 신호어를 찾는다
> `because / so / therefore / thus / hence / as a result`
> + 오늘 배운 단어들: `trigger / contribute / facilitate / result in` 등

**Step 2** — 방향을 확인한다
> `stem from` / `derive from` / `result from` → **뒤가 원인**
> `trigger` / `cause` / `induce` / `result in` → **뒤가 결과**

**Step 3** — 선택지에서 인과 관계가 뒤집혀 있으면 오답!
