# 목적: 긴 문장을 쉬운 의미 단위로 해체해서 독해 속도를 높임
## 앞면: 문장 해체 작업 지시서

[핵심] "누가 / 뭐했다"가 전부다! 나머지는 다 찌끄레기(양념)야!

✂️ 1. 동사(V) 찾고 바로 뒤에서 썰어라!
    - 동사는 문장의 심장이다. 수동태(be+pp), 진행형(be+ing), 완료형(have+pp) 은
      사이에 딴 놈이 껴도 무조건 '한 세트'로 묶어라!
✂️ 2. 쉼표( , ) 보이면 썰어라!
    - 이건 출제자가 준 '숨 쉬는 타이밍(스피드 방지턱)'이다. 무조건 멈춰!
✂️ 3. 전치사(in, on, to 등) 앞에서 썰어라!
    - 여기서부터는 장소/시간 알려주는 '양념' 시작이다. 파울 라인 밟지 마!
✂️ 4. 접속사(and, but, because 등) 앞에서 썰어라!
    - 새로운 문장(레인)이 시작된다는 신호탄이다!
✂️ 5. that / wh- / how 앞에서 무조건 썰어라!
    - 문장 길어지게 만드는 끝판왕들이다. 보이면 묻지도 따지지도 말고 칼질!


## 뒷면: 길 잃지 마라! - 정보의 이정표 24

📍 PART 1. 전치사 12 (명사 앞에 붙는 화살표)
   - TO(→도착) / FROM(←출발) / AT(●지점)
   - IN(📦공간) / ON(👟접촉) / FOR(🎁목표)
   - WITH(🤝동행) / OF(🔗연결) / ABOUT(🌀주변)
   - BY(🏃옆) / UP·DOWN(↑↓수직) / OUT(🚫이탈)
   ※ 전치사 보이면? 무조건 그 앞에서 썰어라(✂️)!

🌉 PART 2. 접속사 12 (문장을 잇는 12개 다리)
   - AND(➕추가) / BUT(🚫반전) / OR(❓선택) / SO(➡️결과)
   - BECAUSE(📢이유) / WHEN(⏰때) / IF(⚠️만약)
   - ALTHOUGH(😮반전주의) / WHILE(🎥동시) / AS(🔄때문에/때)
   - BEFORE(⏪전) / AFTER(⏩후)
   - ※ 접속사 보이면? 새로운 '누가 뭐했다' 시작이다!
